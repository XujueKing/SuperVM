### 详细说明：

#### 📊 三种场景对应不同TPS

1. **高性能场景 (推荐)**: **800K - 1.2M TPS** 🚀
   - 对应测试: Mode 3 多线程并发批量 (1.19M TPS)
   - 适用: DeFi、NFT、GameFi、支付网络
   - 配置: 8-16核 + 并发客户端 + 批量聚合

2. **标准场景**: **400K - 600K TPS** 📈
   - 对应测试: L0.8.1 并行读校验 (495K TPS)
   - 适用: 企业联盟链、供应链、中小DApp
   - 配置: 4-8核 + 单线程优化

3. **保守场景 (兜底)**: **300K - 400K TPS** 🛡️
   - 对应测试: 优化前串行 (318K TPS)
   - 适用: 初期上线、稳定性优先
   - 配置: 标准服务器 + 串行处理

### ❌ 不要用的：

**单线程批量** (171K TPS) - 仅供测试对比，生产环境禁用！
